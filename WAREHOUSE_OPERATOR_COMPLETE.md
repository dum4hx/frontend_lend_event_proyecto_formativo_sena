ADMIN SIDE\n   Admin logs in to /admin/team\n   → Clicks \"Invite User\"\n   → Enters warehouse operator email\n   → Selects \"Warehouse Operator\" role\n   → System generates invite token\n   → Email sent with activation link\n\n2. WAREHOUSE OPERATOR SIDE\n   Opens email\n   → Clicks activation link: /accept-invite?email=...&token=...\n   → Password setup page loads\n   → Sets password (8+ chars, uppercase, lowercase, number, special char)\n   → Confirms password\n   → Account activated successfully\n   → Redirected to login page\n\n3. LOGIN\n   Enters email address\n   → Enters password\n   → Credentials validated\n   → JWT token generated\n   → Auth verified\n   → System checks role: warehouse_operator\n   → Uses getDashboardUrlByRole() → /warehouse-operator\n   → Automatically redirected to dashboard\n\n4. DASHBOARD ACCESS\n   RequireRole guard verifies warehouse_operator role\n   → If authorized: loads WarehouseOperatorLayout\n   → If not authorized: redirects to /unauthorized\n   → Warehouse operator sees their dashboard\n\n5. OPERATIONS\n   Can navigate between 6 sections\n   → Manage inventory\n   → Track locations\n   → View movements\n   → Respond to alerts\n   → Adjust settings\n   → Logout anytime\n```\n\n---\n\n## 📁
